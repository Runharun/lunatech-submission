A minimal web application to persist HTML form submissions.

Use any HTML form to submit data to this URL using a POST request. Use the recent entries page to view the last few results, or export all entries as JSON.

Demo

Submit without redirect: <form method="post" action="/">

Submit with redirect: <form method="post" action="/?redirect=http%3A%2F%2Fexample.com%2Fthank-you">
